# Sajiya Nazir - Frontend Developer Portfolio

A modern, classic UI/UX portfolio website showcasing 10 JavaScript projects.

## Features

- Clean, minimal, professional design
- Responsive layout (mobile + desktop)
- Smooth animations and hover effects
- Dark mode toggle
- Smooth scrolling navigation
- 10 functional JavaScript projects

## Projects Included

1. **Dynamic Text Addition** - Add paragraphs to a div using JavaScript
2. **Color Picker Background** - Change page background with color input
3. **Digital Clock** - Live clock with date and time in 12-hour format
4. **Calculator** - Basic arithmetic calculator
5. **Image Slider/Carousel** - Auto-changing image carousel with controls
6. **API Data Fetcher** - Fetch and display user data from JSONPlaceholder API
7. **Timer App** - Countdown timer with user input
8. **Traffic Signal Simulator** - Animated traffic lights using setTimeout/setInterval
9. **Registration Form** - Form with validation, captcha, and password toggle
10. **Todo List** - Add, edit, and delete tasks with completion tracking

## Technologies Used

- HTML5
- CSS3 (with modern features like CSS Grid, Flexbox, animations)
- JavaScript (ES6+)
- Google Fonts (Poppins)

## How to View

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Navigate through the portfolio and explore each project
